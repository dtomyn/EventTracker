Heatmap Layout Direction

How should the calendar grid be oriented?

A. GitHub-Style Horizontal

Weeks as columns, days of week as rows. One row per year. The classic, instantly recognizable pattern. Scrolls horizontally if many years.

B. Calendar Month Grid

Mini calendar blocks for each month arranged in a row. More compact per year, familiar calendar layout. Each year is a row of 12 mini month grids. Cells represent individual days within each month.
